# Privacy Statement (Public MVP)

Nyxa Governed Memory MCP is designed for operator-controlled deployment and governance workflows.

## Public MVP Privacy Position

- No biometric processing by default.
- No dream interpretation features in public MVP.
- No attachment/personality-kernel features in public MVP.
- No autonomous memory confirmation by default.

## Operator Responsibility

Operators are responsible for lawful deployment, consent handling, and policy configuration in their own environment, including:
- legal basis
- retention windows
- export/delete workflows
- access control
- purpose limitation

## Data Handling Intent

The system is intended to support classification, documentation, and auditability. It is not a legal/clinical decision engine.
