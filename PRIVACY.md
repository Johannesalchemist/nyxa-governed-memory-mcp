# Privacy Statement (Public MVP)

Nyxa Governed Memory MCP is designed for operator-controlled deployment.

## Public MVP Privacy Boundaries

- no default biometric processing
- no dream interpretation in public MVP
- no attachment/personality-kernel features in public MVP
- no autonomous confirmation of memory by default

## Operator Responsibility

Operators are responsible for lawful deployment and consent handling in their environment, including:
- legal basis
- retention and deletion controls
- access control
- purpose limitation

## Design Intent

The system is intended for classification, governance documentation, and audit timelines. It does not replace legal, clinical, or psychological judgment.
